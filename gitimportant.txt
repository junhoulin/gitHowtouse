最一開始的設定

//設定全域用戶名
git config --global user.name "your_name"
 
//設定全域用戶信箱位址
git config --global user.email "your_name@example.com"

//檢視目前config檢視的東西
git config --list 按Q可以跳出來

再來專案引入

//git引入初始化
git init

//檢視檔案狀態
git status

//加入暫存區
git add 檔案名
git aded . 

//存檔
git commit -m "備註"

//查看存檔訊息
git log

//推到github
git push
